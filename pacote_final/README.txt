How to run:
1) Baixar o dataset https://drive.google.com/drive/folders/16kCVi1XH165MrqdV1PawyDfYvxSOVjio?usp=sharing
2) Criar um subdiretório chamado "dataset" na mesma pasta onde está o Jupyter Notebook:
	$ mkdir dataset
3) Entrar na pasta criada no passo (2) e descompactar o arquivo baixado:
    $ cd dataset
    $ unzip <Downloaded file>
3) Abrir o Jupyter Notebook
4) Executar
